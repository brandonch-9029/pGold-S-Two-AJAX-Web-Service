The Beginner's Guide to Art History
API Documentation

GET /periods

Returns a dataset containing each of the featured art periods, their key artists, and information about those artists.
Data returned in JSON list of objects, with each object in this format:

"Period Name": [ {
"timeperiod":"xxxx-xxxx",
"keyinfo": "Information",
"keyartists": [
{"name":"Artist 1","info":"Information"},
{"name":"Artist 2","info":"Information"},
{"name":"Artist 3","info":"Information"}
] } ]

Endpoint URL

http://localhost:3000/periods

GET /artlinks

Returns a dataset containing links to images of paintings for each featured art period.
Data returned in JSON list of objects, with each object in this format:

"Period Name": [ {
"periodWork": [
{"name":"Main Period Work","link":"http://localhost:3000/static/images/period/Period Name.jpg"} ],
"artistWork": [
{"name":"Artist 1's Work","link":"http://localhost:3000/static/images/artist/Period Name-1.jpg"},
{"name":"Artist 2's Work","link":"http://localhost:3000/static/images/artist/Period Name-2.jpg"},
{"name":"Artist 3's Work","link":"http://localhost:3000/static/images/artist/Period Name-3.jpg"}
] } ]

Endpoint URL

http://localhost:3000/artlinks

POST /submit

Submits a user's form input to the server for recording
The data is then written into a text file called responses.txt on the server side.
Data sent as a JSON object in this format:

{
"userName": "John Smith",
"userEmail": "johnsmith@gmail.com",
"userMessage": "Hi, here is my question!"
}

Endpoint URL

http://localhost:3000/submit
